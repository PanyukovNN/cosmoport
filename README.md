# Cosmoport
#### JavaRush test task for online internship

###### This is a web application that keeps records of spaceships in future.

------------


#### Used tecnologies:
1. Maven
2. Tomcat 9
3. Spring
4. Spring Data JPA
5. MySQL
6. MVC

------------


#### Supported operations:
- Get ships list (with filtres)
- Get ships count
- Ordering ships
- Create ship
- Edit ship
- Delete ship
